# Pretext Experiment

Experimenting with `@chenglou/pretext` otherwise known as the future of css to make a unreadable but really really demo of smooth 60fps text wrapping.

## Developer Setup

Clone this repository and run `pnpm install` followed by `pnpm dev` this is a vite app so it should just work.

## Deployment

Deployed automatically by cloudflare pages to `https://pretext-experiments.abelieavablefiction.net`.
